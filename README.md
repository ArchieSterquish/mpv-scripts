# MPV-scripts
## chapterskip.lua [po5](https://github.com/po5/chapterskip)
Skips characters based on regex pattern. New options could be added in chapterskip.lua file. You can specify which options to skip in chapterskip.conf  
## crop.lua [kism](https://github.com/kism/mpvscripts)
Crops video according to aspect ratio. (Implementation of VLC crop button)
## Mpv-menu-plugin (Windows only) [tsl0922](https://github.com/tsl0922/mpv-menu-plugin)
Menu that can be called anywhere on the video. Usage example is in input.conf. 

List of plugin files:
- dialog
- dyn_menu.lua
- menu.dll
## play_next_or_previous_file.lua [jonniek](https://github.com/jonniek/mpv-nextfile)
Plays next or previous file in order
## remember_bluetooth_latency.lua [fatihkaan22](https://github.com/fatihkaan22/mpv-bluetooth-av-delay)
Saves bluetooth latency for wireless headphones
## remember_volume.lua [blackarcher21](https://gist.github.com/blackarcher21/162dc1bef708e90082c6c4f9500c1997)
Saves last used volume
## smart_copy_paste.lua [Eisa01](https://github.com/Eisa01/mpv-scripts/blob/master/README.md#smartcopypaste)
Gives mpv the capability to copy and paste
## smart_subtitle_selector.lua [allecsc](https://github.com/allecsc/mpv-qol-scripts?tab=readme-ov-file#-smart-subtitle-selector)
Selects subtitle based on its name. Can be configured in smart_subs.conf

For example if you have 'Songs' and 'Full' subtitles it'd choose 'Full'
## webm_maker.lua [ekisu](https://github.com/ekisu/mpv-webm)
Gives mpv the cability to create short videos from video without using any external programs. Videos are saved in user 'Videos' directory. This can be changed in webm.conf
## toggle_osc_visibility.lua   
Changes osc visibility between 'never' and 'always'
